# ENVDIFF

**Environment Variable Diff Tool**

## Features

- 🚀 Fast and efficient
- 📦 Single binary, no dependencies  
- 🔧 Easy to use
- 🔒 Secure by design

## Installation

```bash
git clone https://github.com/simplestar-992/envdiff.git
cd envdiff
go build -o envdiff .
```

## Usage

```bash
./envdiff --help
```
