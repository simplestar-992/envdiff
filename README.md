## ✨ Features

- 🚀 Blazing fast performance
- 💎 Clean and intuitive API
- 🔧 Zero configuration needed
- 🛡️ Secure by default
- 📦 Single binary, no dependencies

## 📥 Installation

```bash
git clone https://github.com/simplestar-992/envdiff.git
cd envdiff
go build -o envdiff .
./envdiff --help
```

## 🚀 Quick Start

```bash
./envdiff -h
```

## 🧪 Examples

```bash
# Example 1
./envdiff status

# Example 2  
./envdiff --verbose

# Example 3
./envdiff help
```
