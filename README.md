## Envdiff

A useful command-line tool for developers.

### Installation

```bash
git clone https://github.com/simplestar-992/envdiff.git
cd envdiff
go build -o envdiff .
```

### Usage

```bash
./envdiff --help
```

---

## Quick Start

```bash
./envdiff [command]
```
